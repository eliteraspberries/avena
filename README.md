Requirements
------------

Avena requires:

 -  [Python][] (version 2 or 3);
 -  the [Python Imaging Library][] (PIL) or [Pillow][];
 -  [NumPy][];
 -  and [docopt][].


Installation
------------

Install Avena with [pip][]:

    pip install avena
