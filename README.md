Avena is an image processing library and command-line tool.

[![](https://travis-ci.org/eliteraspberries/avena.svg)][build-status]


Requirements
------------

Avena requires:

 -  [Python][] (version 2 or 3);
 -  the [Python Imaging Library][] (PIL) or [Pillow][];
 -  [NumPy][];
 -  and [docopt][].
