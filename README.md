Install Avena
-------------

Avena requires:

 -  [Python][] (version 2 or 3);
 -  the [Python Imaging Library][] (PIL) or [Pillow][];
 -  [NumPy][];
 -  and [docopt][].

Install Avena with [pip][]:

    pip install avena


Features
--------

 - [x] Flipping
 - [x] Spacial filtering
 - [x] Interpolation
 - [x] Tiling
 - [x] Cross-correlation
